Apa Kata Mereka Tentang Kami

Saya adalah seorang Agent yang bergabung dengan Ferdy Property dan mulai menggunakan website Rumahku.com sejak tahun 2011. Saya senang menggunakan website Rumahku.com karena respon terhadap iklan saya lumayan baik, lalu refresh propertinya mudah, mengedit iklan juga mudah.
Motivasi saya menjadi seorang Agent adalah keuntungan yang akan saya dapatkan, lalu kebebasan waktu, kebebasan berfikir, dan passion. Media yang biasa saya gunakan untuk memasarkan properti saya adalah Koran, Internet, Spanduk, dan Friend to Friend. Pasti ada suka duka ketika menjadi agent. Sukanya ketika saya menjadi seorang Agent adalah bertambahnya relasi dan saya dapat bertukar berbagai informasi dengan cust saya. Dukanya adalah jika penjual dan pembeli yang sudah dipertemukan, melakukan transaksi tanpa mengikut sertakan kita sebagai agentnya, atau dengan kata lain “main belakang”.
Saran saya agar kedepannya website Rumahku.com lebih sering mengadakan Gathering antar Agent. Agar para Agent dapat mengetahui tentang promo yang ada di website Rumahku.com.

Kamis, 30 April 2016

Lita Anggriani

Independen Agent
